WireGuard subnet reference

Citadel Shield uses the 10.100.0.0/24 subnet by default. This gives you 253 usable device addresses.

Server gateway: 10.100.0.1
First device: 10.100.0.2
Last device: 10.100.0.254
Subnet mask: 255.255.255.0 (/24)
Protocol: WireGuard (UDP 51820)
